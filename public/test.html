AI 기반 시선 분석·IR 카메라 융합 기술 사업 추진체계

오프라인 광고의 유효 시청 판별 기술 고도화 및 데이터 기반 광고 플랫폼 완성

사업기간
2026.04.01 ~ 2026.11.30
최종 목표
다양한 환경에서도 안정적으로 동작하는 시선 기반 유효 시청 분석 기술을 확보하고, 이를 광고 운영에 활용 가능한 수준의 데이터 기반 플랫폼으로 완성

추진전략

1

실환경 기반
기술 고도화

실내·실외 환경의 조도, 거리, 각도 조건을 반영하여 실제 적용 가능한 수준으로 기술을 고도화

2

IR 기반
저조도 대응

RGB 카메라 한계를 보완하기 위해 IR 카메라를 도입하고, 야간·저조도 환경 성능을 확보

3

유효 시청
판단 기준 정립

시선 방향과 체류시간을 결합하여 실제 광고 시청 여부를 판단하는 기준을 정립

4

데이터 기반
플랫폼 확장

수집 데이터를 광고 송출, 성과 분석, 리포트 제공까지 연결되는 서비스 구조로 확장

단계별 추진체계

1. 시선 기반 유효 시청 분석 기술 고도화

  • 얼굴 방향 및 시선 분석 알고리즘 개선
  • 정면 여부 판단 정확도 향상
  • 체류시간 기반 유효 시청 판별 로직 고도화
  • 동일 인물 중복 제거 및 추적 기술 개선
›

2. 저조도 환경 대응 IR 기반 기술 개발

  • IR 카메라 기반 얼굴 검출 기술 적용
  • RGB 대비 저조도 환경 성능 비교
  • 야간 환경 시선 분석 정확도 확보
  • 조도별 테스트 데이터 확보
›

3. 환경별 성능 검증 및 데이터 확보

  • 조도, 거리, 각도 변화 조건별 검증
  • 실제 매장 및 상업시설 환경 실증
  • 테스트 데이터 기반 반복 성능 개선
  • 현장 적용 안정성 검토
›

4. 데이터 분석 및 플랫폼 연계

  • 시청 데이터 수집·저장 구조 개선
  • 광고 성과 분석 알고리즘 개발
  • 대시보드 기반 시각화 기능 개선
  • 광고 송출과 데이터 분석 연동 구조 구축
›

5. 실증 및 사업화 기반 확보

  • 실제 운영 매장 환경 적용 및 검증
  • 광고 성과 리포트 생성 및 분석
  • 광고 운영 최적화 시나리오 도출
  • 서비스 확장 기반 구축
